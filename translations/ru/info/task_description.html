На входе вы получаете расписание самолётов в виде {% if is_js %}массива{% else %}списка{% endif %}, каждый элемент которого — это цена прямого воздушного соединения двух городов ({% if is_js %}массив{% else %}список{% endif %} из 3 элементов: первые два — названия городов в виде строк, и третий — цена перелёта).

Самолёты летают в обе стороны и цена в обе стороны одинаковая. Есть вероятность, что соединения между городами может и не быть.

Найдите цену самого дешёвого перелёта для городов, которые переданы 2-м и 3-м аргументами. Если перелет невозможен, верните 0

Входные данные: 3 аргумента: расписание перелётов в виде {% if is_js %}массива массивов{% else %}списка списков{% endif %}, город вылета, город назначения как строки.
