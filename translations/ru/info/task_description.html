"Нам нужно долететь домой как можно дешевле, чтобы больше денег осталось на подарки. Тетя Лида просила сыров разных, а Вася хотел машинку новую. Я уже довольно долго смотрю на расписание и мне начинает казаться, что некоторые самолеты летают зря".

На входе вы получаете расписание самолетов в виде массива, каждый элемент которого это цена прямого воздушного соединения 2х городов (массив из 3х элементов - 2 названия города, в виде строки, и цена перелета).

Самолеты летают в обе стороны и цена в обе стороны одинаковая. Есть вероятность, что соединения между городами может и не быть.

Найдите цену самого дешевого перелета для городов, которые переданы 2ым и 3им аргументами.

Input: 3 аргумента. Расписание перелетов в виде массива массивов. Город вылета и город назначения.

Output: Int. Лучшая цена.

Как это используется: Может быть использовано в повседневной жизни для нахождения оптимальной комбинации.

Precondition: Цена всегда int. В расписании рейсов есть хотя бы один элемент. Оба искомых города есть в расписании.
